# online-tools: tools.holmcc.com
This repository contains the source code for a collection of quick, easy-to-use, web-based tools hosted at `tools.holmcc.com`. The project aims to provide utility tools like clocks, timers, and message displays, with plans for expansion and subtle monetization.

For a detailed development plan, including the tech stack, initial tool set, monetization strategy, and future tool ideas, please refer to the [Project Plan in docs/project_plan.md](docs/project_plan.md).
