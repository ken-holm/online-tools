# Online Tools Suite: tools.holmcc.com

This repository contains the source code for `tools.holmcc.com`, a growing collection of quick, easy-to-use, web-based utility tools. Designed for modern productivity and convenience, the platform offers a seamless experience across devices.

## Features

*   **Comprehensive Toolset:**
    *   **Clock:** Current time and date with 12/24h toggle.
    *   **Countdown Timer:** Set custom events with audio/browser notifications.
    *   **Timer:** Custom duration countdowns with audio/browser notifications.
    *   **Message Tab:** Full-screen "billboard" text display.
    *   **Stopwatch:** Precise timing with lap functionality.
    *   **Breathing Box:** Guided 4-7-8 breathing exercise with visual animation.
    *   **Pomodoro Timer:** Productivity timer with work/break cycles and notifications.
    *   **World Clock:** Track time across multiple timezones with customizable labels.
    *   **Simple Calculator:** Basic arithmetic with keyboard support.
    *   **Password Generator:** Create strong, customizable passwords with strength meter.
    *   **QR Code Generator:** Create custom QR codes with color options and download.
    *   **Text Utilities:** Word/character count, case conversion (Uppercase, lowercase, camelCase, snake_case, kebab-case).
    *   **Unit Converter:** Convert various units (length, weight, temperature, volume) with a "Pro Mode" for esoteric units.
    *   **JSON Formatter:** Prettify, minify, and validate JSON with syntax highlighting.
    *   **Markdown Viewer:** Real-time Markdown editor and previewer with GFM support.

*   **Core Enhancements:**
    *   **Progressive Web App (PWA):** Installable on any device for an app-like experience.
    *   **Search Engine Optimization (SEO):** Dynamic meta tags for improved discoverability.
    *   **Customizable Themes:** Personalize backgrounds and typography via settings.
    *   **Shareable URLs:** Pre-configure tools via custom links.
    *   **Rich Sound Library:** Multiple alarm sound options for timers.
    *   **Keyboard Navigation:** Quick menu access and tool selection.
    *   **Monetization:** Subtle "Buy me a Coffee" and affiliate links.
